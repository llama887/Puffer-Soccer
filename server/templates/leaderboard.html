How it works

  1. Train a soccer policy using the Puffer Soccer environment
  2. Export your policy as a TorchScript module (.pt file)
  3. Upload it here — we run a round-robin tournament against all other policies
  4. ELO ratings update based on head-to-head match results

Policy format: Your .pt file must be a TorchScript module that takes observations of shape (N, {{ 16 + 14 * players_per_team }}) and returns a (logits, values) tuple where logits has shape (N, 13).
